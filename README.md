# ノジマ 購入プラン比較ツール（PWA）

パソコン購入時の延長保証・サポート・プランを比較し、合計金額を試算する社内向けツール。
スマホ／iPad の「ホーム画面に追加」でアプリとして使えます（オフライン対応）。

## 公開URL（GitHub Pages）

`https://rxzero1021.github.io/nojima-pc-plan/`

`main` ブランチに push すると数十秒〜数分で自動的に反映されます（共有リンクも自動更新）。

## ファイル構成

| ファイル | 役割 |
|---|---|
| `index.html` | 本体（画面・計算ロジックすべて） |
| `manifest.webmanifest` | PWA設定（アプリ名・アイコン・テーマ色） |
| `sw.js` | サービスワーカー（オフライン対応・キャッシュ） |
| `icon-*.png` | アプリアイコン |
| `nojima-logo.png` / `mascot.jpeg` | ブランドヘッダー画像 |
| `sapo1.jpeg` / `sapo2.jpeg` / `uirusu.jpeg` | クリップで開く案内シート画像 |

## 編集のしかた（VS Code）

1. このリポジトリをクローン、または `pc-plan-app` フォルダを VS Code で開く
2. `index.html` を編集（画面・計算はここに集約）
3. 内容を変えたら `sw.js` の `const CACHE = 'pc-plan-vXX'` の番号を1つ上げる
   （インストール済み端末のキャッシュを更新するため）
4. `git add -A && git commit -m "変更内容" && git push`
5. GitHub Pages が自動でビルド → 公開URLが更新される

## メモ

- 金額計算はすべて税込入力前提。
- 延長保証はPOS売価（各種値引施策適用前・税込）を基準に計算。
- パソコンは22万円以上のとき、POS売価の5%を保証料に加算。
- パソコン以外はライト・スタンダードが5,500円またはPOS売価の5%、プレミアムが11,000円またはPOS売価の10%。
- POS売価が未入力のときは合計を表示しない仕様。
